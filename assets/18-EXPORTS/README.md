# 18-EXPORTS/ — Exports finaux par format

Exports finaux, prêts à l'emploi, **classés par format**. Destinés à être consommés (site, presse, partenaires).

| Dossier | Contenu | Usage |
|---|---|---|
| `PNG/` | Images matricielles à fond transparent / captures. | Web, UI, réseaux. |
| `JPG/` | Photos exportées (sRGB, compressées). | Web, presse. |
| `SVG/` | Vectoriels (logos, icônes, pictogrammes). | Web, impression. |
| `PDF/` | Documents (dossier de presse, planches, one-pagers). | Diffusion, impression. |

## Convention de nommage

`sujet-usage-AAAA-MM-JJ.ext` — ex. `logo-fadi-dahdouh-web-2026-07-01.svg`, `dossier-presse-2026-07-01.pdf`.

## Règles

- Optimiser avant export (poids maîtrisé — la performance est une forme de respect, cf. Experience Bible).
- Conserver le master dans `13-SHOOTS/.../RETOUCHED/` ou `14-BRAND/` ; ici, uniquement le dérivé final.
- Ne jamais éditer un export : repartir de la source.
