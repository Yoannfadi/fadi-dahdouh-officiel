# 17-MOCKUPS/ — Exports de maquettes

Exports plats (images) des maquettes pour revue, partage et archivage. **La source de vérité du design reste Figma** (cf. `docs/PROJECT-MASTER-INDEX.md`) ; ce dossier ne contient que des rendus exportés.

| Dossier | Contenu |
|---|---|
| `Homepage/` | Exports des maquettes desktop (Hero « Le Jardinier », sections d'accueil). |
| `Mobile/` | Exports des maquettes mobile (375). |
| `Social/` | Déclinaisons réseaux sociaux (Instagram, LinkedIn, YouTube). |

## Convention de nommage

`ecran-etat-AAAA-MM-JJ-NN.png` — ex. `hero-jardinier-etat2-2026-07-01-01.png`.

## Règles

- Format : `.png` (revue) ; `.pdf` pour les planches (→ aussi `18-EXPORTS/PDF/`).
- Indiquer la date : un mockup est un instantané, pas la source.
- Ne jamais coder à partir d'un mockup exporté — toujours depuis Figma.
