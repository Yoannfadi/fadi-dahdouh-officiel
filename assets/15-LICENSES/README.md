# 15-LICENSES/ — Licences & droits

**Toutes les licences sont archivées ici.** Aucun asset (police, image, icône, musique) ne peut être utilisé sur le site sans sa licence archivée dans ce dossier.

| Dossier | Contenu |
|---|---|
| `Fonts/` | Licences des polices (commerciales & gratuites) — factures, EULA, étendue d'usage (web/desktop). |
| `Images/` | Licences/cessions de droits des photos, autorisations de droit à l'image des personnes. |
| `Icons/` | Licences des jeux d'icônes. |
| `Music/` | Licences des musiques / habillages sonores. |

## Convention de nommage

`fournisseur-asset-AAAA-MM-JJ.pdf` — ex. `commercial-canela-2026-07-01.pdf`, `cession-portrait-fadi-2026-07-01.pdf`.

## Règles

- Une licence par fichier, format `.pdf` de préférence.
- Toute personne identifiable sur une photo doit avoir une **autorisation de droit à l'image** archivée dans `Images/` (RGPD — cf. `19 — Security`).
- Conserver le justificatif d'achat ET les conditions d'usage.
