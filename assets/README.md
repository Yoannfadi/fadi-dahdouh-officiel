# assets/ — Bibliothèque des médias

Bibliothèque centrale des médias du projet **fadi-dahdouh-officiel**.
**Source de vérité :** le CODEX (`docs/00-CODEX/`) + le FDOS (`docs/PROJECT-MASTER-INDEX.md`).

> Règle d'or visuelle (CODEX) : clair-obscur **Rembrandt**, l'humain, le geste, la matière. **Jamais de banque d'images générique.** La lumière est blanche et chaude ; **l'or `#BFA06E` n'est jamais une lumière**, seulement une ponctuation.

---

Ratios de référence (CODEX) : **portraits 4:5**, **scènes 16:10**, vidéos 16:9.

## 2. Quels fichiers déposer

- **Photos** : `.jpg` (livraison) et/ou `.tif`/`.png` (master) — sRGB, qualité maximale, non recadrées « à l'arrache ».
- **Vidéos** (`11-VIDEOS/`) : `.mp4` (H.264/H.265), + master si disponible.
- **Logos** (`09-LOGOS/`) : `.svg` en priorité, sinon `.png` à fond transparent.
- **Textures** (`10-TEXTURES/`) : `.jpg`/`.png` haute résolution.
- Chaque image doit pouvoir tenir sa place dans l'univers **noir + or, lumière Rembrandt**.

## 3. Photos interdites

- ❌ Banque d'images générique / stock.
- ❌ Lumière à plat, néon, studio visible, contre-jour, sensation publicitaire.
- ❌ Filtres froids, sur-saturation, HDR.
- ❌ Fichiers **non renommés** sortis du boîtier ou d'un export brut :
  - `IMG_4821.jpg`
  - `DSC_9382.jpg`
  - `photo_finale_v8.jpg`
- ❌ Patients/personnes identifiables sans autorisation écrite (discrétion — CODEX).

## 4. Conventions de nommage

Format : `categorie-sujet-traitement-NN.ext` — minuscules, tirets, numéro à 2 chiffres. Pas d'espaces, pas d'accents, pas de majuscules, pas de versions floues (`_v8`, `final`, `def`).

**Exemples valides :**
```
hero-rembrandt-01.jpg
hero-regard-camera-01.jpg
hero-noir-blanc-01.jpg
hero-profil-01.jpg
portrait-editorial-01.jpg
cabinet-paris-01.jpg
```

**Exemples interdits :**
```
IMG_4821.jpg
DSC_9382.jpg
photo_finale_v8.jpg
```

## 5. Règle absolue du Hero

> Le **Hero « Le Jardinier »** ne doit utiliser **que** les images de **`assets/00-HERO/Approved/`**.
> Aucune image de `Alternatives/`, `Rejected/`, ou d'un autre dossier ne peut alimenter le Hero.
