# 00-HERO/ — Images du Hero « Le Jardinier »

Ce dossier contient **uniquement** les images candidates au **Hero**, la référence artistique absolue du site (direction **Le Jardinier**, page Figma `23`).

> **RÈGLE ABSOLUE :** le Hero du site ne doit utiliser **que** les images de **`00-HERO/Approved/`**. Rien d'autre. Jamais `Alternatives/`, jamais `Rejected/`, jamais un autre dossier.

---

## Les trois sous-dossiers

| Dossier | À quoi il sert | Quoi y déposer |
|---|---|---|
| `Approved/` | Les images **validées** par Fadi, prêtes à alimenter le Hero. | Le ou les portraits retenus, finalisés, conformes à la lumière de référence. **C'est la seule source autorisée du Hero.** |
| `Alternatives/` | Pistes sérieuses, non retenues pour l'instant mais conservées. | Variantes valables (autre regard, autre cadrage) gardées en réserve. |
| `In-Progress/` | Travail en cours : tri, retouche, essais. | Images en cours d'évaluation, pas encore tranchées. Rien ici n'alimente le Hero. |
| `Rejected/` | Archive des images écartées. | Tout ce qui a été testé puis refusé — conservé pour mémoire, jamais publié. |

> La fiche complète de la photo retenue (pourquoi, date, photographe, lumière, cadrage, historique…) est tenue dans [`HERO.md`](HERO.md).

## La photo du Hero — cahier de lumière (référence absolue)

Toute image déposée ici doit respecter la lumière **Rembrandt** définie dans le laboratoire (Figma `23`) :

- Une seule source principale, angle **≈ 45°**.
- Une joue éclairée, l'autre dans une **ombre profonde**.
- **Triangle de Rembrandt** perceptible sur la joue à l'ombre.
- Le **fond disparaît dans le noir** — aucun décor.
- Lumière **blanche, chaude, naturelle**. **L'or n'est jamais une lumière.**
- **Interdits :** contre-jour, lumière de studio visible, sensation publicitaire, lumière qui flatte.
- Doit évoquer : **silence · présence · transmission · profondeur**. Un portrait **intemporel**.
- Cadrage cible : **4:5**, le portrait vit à droite, le vide est une matière.

## Quels fichiers déposer

- `.jpg` haute qualité (sRGB) pour la livraison ; master `.tif`/`.png` bienvenu à côté.
- Le portrait doit fonctionner en **deux états** : à peine perceptible (arrivée) puis pleinement révélé (rencontre).

## Photos interdites

- ❌ Banque d'images, lumière à plat / froide / studio visible / pub.
- ❌ Fichiers bruts non renommés : `IMG_4821.jpg`, `DSC_9382.jpg`, `photo_finale_v8.jpg`.
- ❌ Toute image qui « montre » au lieu de « révéler ».

## Conventions de nommage

`hero-sujet-traitement-NN.jpg` — minuscules, tirets, numéro à 2 chiffres.

**Exemples valides :**
```
hero-rembrandt-01.jpg
hero-regard-camera-01.jpg
hero-noir-blanc-01.jpg
hero-profil-01.jpg
```

**Interdits :**
```
IMG_4821.jpg
DSC_9382.jpg
photo_finale_v8.jpg
```
