# DECISIONS — Registre des décisions structurantes (ADR)

Chaque **décision structurante** possède son propre document numéroté (style *Architecture Decision Record*).

## Convention
`NNNN-Titre.md` — numéro à 4 chiffres, incrémental, jamais réutilisé.
Format de chaque ADR :
```
# NNNN — Titre
- Statut : Figé | Accepté | En arbitrage | Remplacé par NNNN
- Date · Décideurs
## Contexte
## Décision
## Conséquences
## Alternatives écartées
## Retour arrière
```
