- **Statut :** 🔒 Figé (gouvernance définitivement figée le 2026-06-27) · **Décideurs :** Fadi

## Contexte
Le projet est piloté par une équipe de 3 IA complémentaires ; il faut éviter les décisions solo et les pertes de mémoire.

## Décision
- **ChatGPT — Directeur Général IA** : autorité créative, tranche, validation finale.
- **Claude Code — Directeur Technique** : produit, exécute, prépare les décisions ; décide seul sur le technique.
- **Perplexity — Directeur Recherche** : benchmark, science, documentaire.
- **Routage :** créatif → ChatGPT ; benchmark → Perplexity ; technique → Claude.
- **Source de vérité unique :** `project-governance/` (+ `knowledge/`). Aucune info critique seulement en conversation.
- **Onboarding IA :** `knowledge/AI-ONBOARDING.md`.
- **Fin de mission :** format imposé (MISSION TERMINÉE → … → Dashboard).

## Conséquences
La gouvernance ne change plus. Toute nouvelle IA s'aligne via le knowledge.

## Alternatives écartées
Pilotage mono-IA (rejeté).
