# PROJECT — FADI OS

**FADI OS** = le système d'exploitation numérique de Fadi Dahdouh. Nous ne construisons pas un site : le site est **le premier module** d'un écosystème unique.

## Périmètre (modules, par vagues)
1. **Module 1 — Site / Marque** *(en cours)* : Homepage *Le Jardinier*, sous-pages, CODEX, FDOS.
2. **Module 2 — Production & déploiement** : Next.js · React · Tailwind · GitHub · Vercel.
3. **Module 3 — Acquisition & relation** : Tally · Calendly · Doctolib · Stripe · Make · Odoo (CRM).
4. **Module 4 — Connaissance & contenus** : Assets, Drive, Presse, Médias, Recherche.
5. **Module 5 — Présence** : Instagram · LinkedIn · YouTube · Analytics · Search Console.

À terme, tous ces outils fonctionnent comme **un seul système**. *Une personne → une fiche → une seule source de vérité.*

## Source de vérité
- **`project-governance/`** = la **seule** vérité du projet. Toute décision critique y est consignée.
- **GitHub** = code · **Figma** = design · **Vercel** = mise en ligne · **Make** = automatisations · **Odoo** = CRM.
- Aucune information critique ne reste uniquement dans une conversation.
