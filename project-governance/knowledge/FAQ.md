# FAQ

**Q. C'est un site ?**
Non. C'est **FADI OS**, un système d'exploitation numérique. Le site est le module 1.

**Q. Quelle est la phrase d'accroche ?**
« On cherche la voix. On trouve la personne. » — gelée, ne pas modifier.

**Q. Quelle est la structure de la Homepage ?**
6 blocs figés : Hero → Silence → Voix du Fondateur → Trois Chemins → Transformation → Invitation.

**Q. Pourquoi pas de bloc « Méthode » ni de grille de preuves sur la home ?**
Pour préserver l'émotion (montrer > expliquer). La méthode va en sous-page « L'Approche », les références en « Reconnaissance ».

**Q. Qui décide quoi ?**
ChatGPT tranche le créatif (DG IA), Claude exécute le technique, Perplexity documente la recherche.

**Q. Peut-on ajouter une section / un bouton / un logo ?**
D'abord se demander ce qu'on peut **retirer**. Pas de discours commercial, pas de grille de logos, pas de bullet points.

**Q. Pourquoi le Hero n'est pas encore « en production » ?**
Il manque le **fichier HD + la licence Olivier Elliard** pour la photo (« Hero Approved V1 »).

**Q. Où est la vérité du projet ?**
`project-governance/` (opérationnel) + `project-governance/knowledge/` (mémoire permanente). Jamais uniquement dans une conversation.

**Q. Le branding du Google Drive ?**
Obsolète. La seule source = le repo local (CODEX).

**Q. On travaille comment ?**
Par lots (batch) : plan → production → validation → mission suivante. Voir HOW-WE-WORK.md.
