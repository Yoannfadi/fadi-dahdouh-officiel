# VALUES

## Les 4 verbes
**Comprendre · Protéger · Révéler · Transmettre.**

## Mantras
- *« Révéler. Pas fabriquer. »*
- *« Celui qui transmet ne meurt jamais. »*

## Archétype
**Le Révélateur** — incarné par la métaphore du **Jardinier** : il crée les conditions, il n'impose rien ; il aide à éclore ce qui était déjà là.

## Postures
- **Présence > performance.**
- **Révélation > démonstration.**
- **Transmission > expertise.**
- **Humain avant technique.**
- **Le silence est une matière ; le vide est un outil.**

## Règle d'or (toute décision)
1. *Est-ce que cela donne davantage envie de rencontrer Fadi Dahdouh ?* — sinon, supprimé.
2. *Que peut-on retirer ?* — avant *que peut-on ajouter ?*
3. *Le visiteur a-t-il l'impression de rencontrer un être humain, pas de visiter un site ?*

## Ton
Sobre, intemporel, souverain. Jamais commercial, jamais explicatif trop tôt. Registre Cucinelli / Aesop / Leica.
