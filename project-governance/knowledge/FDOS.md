# FDOS — Fadi Digital Operating System

« On ne construit pas un site, on construit un système. » Le site est **le module 1**.

## Modules (par vagues)
1. **Site / Marque** — Homepage *Le Jardinier*, sous-pages, CODEX. *(en cours)*
2. **Production & déploiement** — Next.js · React · Tailwind · GitHub · Vercel.
3. **Acquisition & relation** — Tally · Calendly · Doctolib · Stripe · Make · Odoo (CRM).
4. **Connaissance & contenus** — Assets, Drive, Presse/Médias, Recherche.
5. **Présence** — Instagram · LinkedIn · YouTube · Analytics · Search Console.

## Principe central
**Une personne → une fiche → une seule source de vérité.** Tous les outils convergent vers un système unique (orchestré par Make, mémoire dans Odoo + `project-governance/`).

## Couche IA
ChatGPT (direction) · Claude (technique) · Perplexity (recherche). L'IA **transmet une mémoire, pas une présence** : les validations narratives/cliniques restent humaines.

## Convention d'état
✅ VALIDÉ · ◐ À VALIDER · ○ À DÉFINIR. Les données sensibles ne sont **jamais inventées** (emplacements vides balisés).
