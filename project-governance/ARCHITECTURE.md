# ARCHITECTURE

## 1. Architecture narrative (Homepage — figée)
PROMESSE → CROYANCE → AMPLEUR → PREUVE → INVITATION. Les blocs **n'apparaissent pas, ils émergent** les uns des autres (noir = matière commune, lumière = fil conducteur, vide = rythme).

1. **Hero** — *On cherche la voix. On trouve la personne.* (portrait 007)
2. **Le Silence** — respiration, photo pleine page, zéro texte.
3. **La Voix du Fondateur** — 4-6 lignes, présent, registre Cucinelli/Aesop/Leica. Objectif : confiance.
4. **Les Trois Chemins** — SOIN · CRÉATION · TRANSMISSION (une image + une phrase chacun). Objectif : ampleur.
5. **Une Transformation** — une seule histoire humaine. Objectif : preuve.
6. **L'Invitation** — une porte, pas une injonction. Objectif : ouverture.

**Hors colonne (sous-pages) :** « L'Approche » (méthode), « Reconnaissance » (logos/médias/références).

### Trois Chemins → entités
- **SOIN** : Ostéopathie de la voix.
- **CRÉATION** : Art Vox · CMC Studio.
- **TRANSMISSION** : Executive Program · Formation · LSV.
- *(PolymniVox = recherche → à rattacher à TRANSMISSION — à confirmer, voir RISKS.)*

## 3. Écosystème FDOS (cible)
Site (Vercel) → Tally (forms) → **Make** (orchestrateur) → Calendly / Doctolib (RDV) → Google Calendar · Stripe (paiement) · **Odoo (CRM, source de vérité ventes)** · Gmail/Contacts/Drive · Analytics/Search Console · Meta-Instagram/LinkedIn/YouTube.
**Principe :** une personne → une fiche → une seule source de vérité.
