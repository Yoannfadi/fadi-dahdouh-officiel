# QUALITY — Checklist avant toute validation

Claude Code vérifie systématiquement, avant de présenter un livrable :

- [ ] **Cohérence CODEX** — tokens couleur, typo, base-8, Rembrandt, tempo largo.
- [ ] **Cohérence FDOS** — la pièce s'inscrit dans le système (pas un objet isolé).
- [ ] **Cohérence UX** — clarté, hiérarchie, friction maîtrisée, parcours.
- [ ] **Cohérence narrative** — prolonge l'émotion du Hero, aucune rupture de registre, les blocs émergent.
- [ ] **Cohérence graphique** — noir/vide/lumière/clair-obscur, l'or jamais une lumière, photo documentaire.
- [ ] **Cohérence technique** — faisable proprement en Next.js/Tailwind, composable, responsive.
- [ ] **Cohérence SEO** — structure sémantique, titres, métadonnées (au dev).
- [ ] **Cohérence responsive** — pensé mobile dès la conception.
